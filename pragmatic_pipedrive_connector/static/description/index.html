Odoo PipeDrive Connector

At Pragmatic, we have developed Odoo - Pipedrive Connector which allows easy data flow between Pipedrive to Odoo and vice versa.

*Note:
1] This particular app applies for default odoo configuration settings.
2] Ensure that Stage Names of Deals are not changed.

Step 1:

To provide pipedrive URL’s:

Settings->General Settings->Pipedrive Connector
Set Pipedrive Authorize URL: https://oauth.pipedrive.com/oauth/authorize
Set Pipedrive Token URL:https://oauth.pipedrive.com/oauth/token
Set Pipedrive API URL:https://api-proxy.pipedrive.com/

Step 2:

To get connected with pipedrive.

  • Settings->Users & companies->My Company
  • Click on tab->Pipedrive connector
  • Fill in the required details:

1. To get Client ID and Client secret

  • ->Login into pipedrive account with valid credentials
  • ->Click on Settings towards the right side
  • ->In app details,You can find is Client ID and Client Secret
  • Client Secret is hidden by default,Click on Show so that it gets displayed.

    Click on Authenticate button

    • This will redirect you and the following will be displayed.You need to install the app for the first time.If installed,You get the following
    • Click on continue to the app.
    • If successful,You get a message:”You can close this window now”. Else,check if you have entered all the details properly.

Step 3:

You are now connected with pipedrive and do all that you want easily!!!

1. Let’s start with Importing organization:
Click on a button named Import Organization.This button will import(get) all organizations from Pipedrive to Odoo.

The very first thing is you need to add organization in Pipedrive.

Access your pipedrive account.

Go to->Contacts->Organisations

Click on Add Organisation and fill in the necessary details.

Fill in the details and click on Save.

If you see above image,pragmatic tech soft pvt lmt is the organisation saved in Pipedrive.To import in odoo,follow the steps mentioned below:

1.Click on Import Organisation.

Your organisation will be imported to odoo.In order to verify,Go to contacts and search for the organisation name of pipedrive.

If you click on Edit and check,the company type preserved will be of type company.

2.Importing Contacts

This will import pipedrive people to odoo contacts.

To do so,click on Contacts ->People

Click on Add person

After Clicking on Save,Contacts is saved in Pipedrive.

To import to odoo,Click on button->Import Contacts

*Points to observe:

  • Company type :individual.
  • Person gets allocated to that particular organisation.

3.Import Products:

Create a product in pipedrive. Products->Add product

Create a product in Pipedrive.After clicking on save,product will be saved in Pipedrive.

To import,Click on Import Products

*Points to be observed:

  1. Product code gets mapped to Internal Reference,as can be seen above.
  2. Pipedrive Direct Cost and currency gets saved into the tab of Pipedrive Products Information.

4.Import Stages

To get deals of pipedrive to odoo,you need to import stages.

Note that:Import stages are required only once.You need not import it again.

Click on Import Stages and proceed further.

5.Import Deals(In Proposal)

This will import the deals that are still in progress in Pipedrive.

Note that you can add an activity as well when creating a deal.

Firstly,create a deal in Pipedrive.

Once you save the deal,click on it and do the further steps:

You can now add activities to your deal. Click on Add activity

Click on Save,the activity gets associated with that particular deal.

You can add more than one activity for a deal.

In order to import deal as well as activity,ensure that you first Click on Import Deals(In proposal) and click on Import Activities.

In order to check if deals and activities are imported to odoo.

Go to->CRM->Leads

In order to get the deal in the pipeline,you just need to click on Convert to opportunity.

To check in pipeline,

Sales->My Pipeline

Points to observe:

Whatever stage the deal is in pipedrive,it remains the same in odoo as well. To get activities,associated with the deal,you just need to scroll down.

Activities associated with piedrive_org deal:

1]Mail activity

2]Call activity

You can also,mark activity as done,which will replicate the same in pipedrive.

Once u click on Mark done,provide a feedback and click on the button “Done” ,which means the activity is performed and needs no reminders.

To check in pipedrive,if that particular activity is marked done,just go to the same deal in pipedrive and refresh the page.

6)Import Won/Lost Deals

This will import deals that are either won or lost in pipedrive.

Create a won deal in pipedrive.Follow the same process as above.Just to mark it as won,click on Won button at the right hand side.

Once won deal is created in pipedrive,Click on Import Won/Lost Deals.

In above won deal, is displayed in light grey which identifies this deal is won.

Follow the same process and just click on Lost to mark the deal as lost.

In order to check out in odoo,apply filter of lost in CRM->Leads
